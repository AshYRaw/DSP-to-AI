Core Structures
IIR Filter (State-Space Form)
Recursive filter with infinite impulse response using internal delay states
h[k] = Āh[k−1] + B̄u[k]
≡
Mamba / SSM Recurrence
State-space model with hidden state updated at each timestep
x_t = Ā_t·x_{t−1} + B̄_t·u_t
Cross-Correlation
Measure of similarity between template and signal at each lag
R(τ) = ∫ r(t)·s*(t−τ) dt
≡
QKᵀ Attention Scores
Pairwise dot-product similarity between query and key vectors
A_ij = q_i · k_j / √d_k
Matched Filter
Optimal SNR detector: convolve signal with time-reversed template
h_mf(t) = s*(T − t)
≡
Self-Attention (Single Head)
Content-adaptive matched filter — templates are learned from data
softmax(QKᵀ/√d_k)·V
Rational Transfer Function
Ratio of polynomials in z — defines all LTI filter behavior
H(z) = B(z)/A(z)
≡
SSM Transfer Function
State-space produces the same rational function
H(z) = C(zI − Ā)⁻¹B̄ + D
Stability & Memory
Pole Locations in z-Plane
Roots of denominator A(z) — determine resonance, decay, stability
|pole_i| < 1 → stable
≡
Eigenvalues of Ā Matrix
Diagonal entries of Ā — each is a pole of the SSM's transfer function
|λ_i| < 1 → BIBO stable
Parallel First-Order Sections
Partial fraction expansion for numerical stability in high-order filters
H(z) = Σ rᵢ/(z − pᵢ)
≡
Diagonal State Matrix (S4D)
Each dimension is an independent 1st-order system
Ā = diag(λ₁,…,λ_N)
BIBO Stability
Bounded Input → Bounded Output. All poles inside unit circle.
≡
SSM Stability Constraint
Parameterize A with negative real parts, discretize: |exp(Δλ)| < 1
Design & Adaptation
Butterworth / Chebyshev Design
Systematic pole placement for desired frequency response shape
≡
HiPPO Initialization
Systematic pole placement for optimal signal history compression
Adaptive Filter (LMS / RLS)
Time-varying coefficients adapting to track changing signal statistics
≡
Mamba's Selective Scan (S6)
Input-dependent B, C, Δ — filter adapts at every timestep via backprop
Mel / Wavelet Filter Bank
Parallel band-pass filters decomposing signal into frequency bands
≡
Multi-Head Attention / D SSM Channels
Parallel learned projections, each capturing different patterns
Computation Modes
FIR Convolution (via FFT)
Finite impulse response — trivially parallelizable, no stability issues
y = h * x → Y = H·X (freq domain)
≡
S4 Training Mode (Convolution)
Unrolled kernel K̄ convolved with input via FFT — parallel training
y = K̄ * u, K̄ = (CB̄, CĀB̄, …)
IIR Recurrence (Sequential)
O(1) per sample, must process sequentially, efficient for streaming
y[n] = Σ b_k·x[n−k] − Σ a_k·y[n−k]
≡
Mamba Inference Mode (Recurrence)
O(1) per step, fixed memory, natural streaming — same IIR structure
x_t = Ā_t·x_{t-1} + B̄_t·u_t

⚡ The Deepest Insight

Attention is a data-dependent FIR filter — it computes explicit weights for every lag (O(n²) parameters for n frames). Mamba is a data-dependent IIR filter — it uses recursive state to implicitly represent all lags (O(N) state for any n). This is exactly why attention scales quadratically and Mamba scales linearly. It's the FIR vs IIR tradeoff, wearing a neural network hat.
