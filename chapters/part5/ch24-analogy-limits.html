Part V — The Critical Lens
Chapter 24

Where Analogies Break

Parts I–IV built elegant bridges between DSP and AI. Now we test them rigorously. Every analogy has a domain of validity — and understanding where they break is just as important as understanding where they hold. This chapter examines the four deepest mathematical challenges to the DSP↔AI correspondence.

24.1 The Matched Filter ↔ Attention Analogy

In Chapter 17, we showed that self-attention evolves from the classical matched filter. The core claim: QKT is a cross-correlation matrix. This is pedagogically powerful — but three compounding nonlinearities challenge the analogy:

1. Softmax normalization

The matched filter computes linear cross-correlation. Softmax attention adds competitive normalization: αij = exp(qi·kj)/Σ exp(qi·kl). The “correlation” at position (i,j) depends on all other positions — this has no classical matched filter analogue.

2. Learned projections

WQ, WK, WV mean the “template” changes via backpropagation. A matched filter with a data-dependent template is no longer a matched filter — it’s a nonlinear estimator.

3. The value pathway

In matched filtering, detection and estimation are one operation. In attention, detection (QKT) and information extraction (weighting V) are decoupled — closer to a two-stage detector-estimator.

The Spectrum of Validity

The analogy holds rigorously in one specific limit: linear attention. Removing softmax gives Attn(Q,K,V) = (QKT)V — a pure linear operator that IS cross-correlation in the projected space. This is actually the true matched filter analogue. Softmax is the crucial departure that gives Transformers their power.

Scaled dot-product (pre-softmax) Normalized cross-correlation Exact

Show the Math — Detection Efficiency Analysis

The matched filter maximizes output SNR under AWGN via the Cauchy-Schwarz inequality:

SNRout = |∫ H(f)S(f)ej2πfTdf|² / (N0/2 · ∫|H(f)|²df)

Define detection efficiency η = SNRattention / SNRmatched filter. As softmax temperature τ varies:

  • τ → 0 (hard argmax): η drops — only one position selected, information lost
  • τ = 1/√dk (standard scaling): η ≈ 0.85–0.95 under AWGN
  • τ → ∞ (uniform weighting): η drops — all positions weighted equally, no discrimination

Under non-Gaussian noise (colored, impulsive, speech-shaped), attention can exceed the matched filter (η > 1) because learned projections adapt to the noise structure. This is where the nonlinearity pays off.

24.2 The IIR ↔ Mamba Analogy

Chapter 18 showed that S4’s state-space model is mathematically identical to an IIR filter: H(z) = C(zI − Ā)−1&Bmacr; + D. But Mamba introduces input-dependent parameters (Bt, Ct, Δt), making the system Linear Time-Varying (LTV) — and this changes everything.

H(z) does not exist

The z-transform transfer function requires time-invariance. For LTV systems, you need the time-varying impulse response h(k, m) — a 2D function. There is no single H(z) for Mamba.

Poles shift continuously

When Āt = exp(Δt · A) changes at every timestep, “poles” become instantaneous eigenvalues that shift with the input. The system may be instantaneously unstable yet globally stable.

BIBO stability differs

For LTI: |λi| < 1 is necessary and sufficient. For LTV, eigenvalue conditions are neither. Stability requires Lyapunov exponents Λ = lim (1/n) log ||Φ(n,0)|| < 0.

The “Frozen Transfer Function” — A Practical Compromise

At any time t, freeze Bt, Ct, Δt to get an instantaneous Ht(z). Watching how Ht(z) evolves over time connects Mamba to short-time Fourier analysis — you’re taking “snapshots” of the filter, exactly like a spectrogram snapshots a signal’s spectrum.

The correct DSP analogue for Mamba is not a fixed IIR filter but a Kalman filter — a time-varying state estimator where gains adapt based on the signal. This is actually a more powerful analogy, since Kalman filters are fundamental in DSP and controls.

New Research: Hidden Attention of Mamba (ACL 2025)

Ali et al. showed that Mamba’s SSM can be rewritten as implicit attention matrices. The time-varying recurrence produces attention-like patterns without explicit QKT computation. This bridges the IIR and attention interpretations — Mamba’s “hidden attention” emerges naturally from its state-space dynamics.

Show the Math — Lyapunov Exponent Analysis

For the time-varying system xk+1 = Ak xk, the maximal Lyapunov exponent is:

Λ = limn→∞ (1/n) log ||Φ(n,0)||
where Φ(n,0) = An-1 An-2 … A0

For Mamba with diagonal A, the instantaneous poles are λi(t) = exp(Δt · ai). Since Re(ai) < 0 by design and Δt > 0 (softplus output), we get |λi(t)| = exp(Δt · Re(ai)) < 1 for all t.

This was formally proven at ICLR 2025: Mamba’s Lyapunov exponents are non-positive by construction, making it stable without additional constraints.

24.3 The FIR/IIR Mapping Is Incomplete

The elegant claim from Chapter 20: “Attention = FIR (finite context, O(n²)), Mamba = IIR (infinite memory, O(n)).” While capturing the essential tradeoff, this mapping has several problems:

KV-cache makes attention recursive

During autoregressive inference, attention maintains a growing KV-cache. Each output depends on the entire cache — this is not FIR behavior. An FIR filter has a fixed-length impulse response; attention’s grows with the sequence.

Mamba’s “infinite memory” is lossy

With state dimension N=16, Mamba has ≈512 bits to represent all past information, regardless of sequence length. By the data processing inequality, I(ht; x1:t) ≤ N · log2(precision). This is fundamentally different from an IIR filter processing a deterministic signal.

O(n²) vs O(n) elides constants

For short sequences (n < 256), attention is often faster because matrix multiplication is supremely optimized on GPUs (tensor cores), while Mamba’s parallel scan has higher constant factors.

Show the Math — Information Bottleneck

By the data processing inequality, the mutual information between Mamba’s state and the input history is bounded:

I(ht ; x1:t) ≤ H(ht) ≤ N · log2(precision) bits

For float32 with N=16, this is at most ~512 bits. In contrast, attention’s KV-cache stores the full sequence with O(n · d · 32) bits. The FIR/IIR framing should be presented as a first-order approximation that captures the essential tradeoff, with explicit acknowledgment of these limits.

24.4 HiPPO ↔ Butterworth: Different Optimization Spaces

Chapter 18 drew a connection: “HiPPO places poles optimized for memory, just as Butterworth places poles optimized for frequency response.” While both involve pole placement, they operate in fundamentally different mathematical spaces.

Butterworth

Optimizes in the frequency domain: minimizes |H(jω)|² deviation from 1 in the passband, subject to monotonicity (maximally flat). Poles are uniformly distributed on a circle in the s-plane, yielding complex z-plane poles.

HiPPO

Optimizes in the approximation theory domain: minimizes weighted L² error of history reconstruction from state coefficients. HiPPO-LegS eigenvalues are at −(n+½) on the negative real axis — all real, nothing like Butterworth’s circle.

The Resolution

The analogy should emphasize the shared methodology (principled pole placement in state-space systems) rather than implying the results are similar. The strongest bridge case: HiPPO-FouT (the Fourier variant) places poles on a circle, which does resemble Butterworth. This variant is the true bridge between the two optimization spaces.
