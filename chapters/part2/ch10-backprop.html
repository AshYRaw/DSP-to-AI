Chapter 10

Neural Networks and Backpropagation

A single neuron draws a line. Stack neurons in layers and you can draw anything. The challenge is training them — finding the millions of weights that make the network useful. Backpropagation solves this by propagating the error backward through the network, one layer at a time, using the chain rule of calculus.

10.1 Multi-Layer Perceptrons

The XOR problem from Chapter 9 showed that a single neuron can only draw linear boundaries. The fix is to stack neurons in layers:

Input Layer

One neuron per feature. No computation — just passes the raw data to the next layer. For 2D classification: 2 inputs.

Hidden Layer(s)

Where the magic happens. Each neuron combines inputs nonlinearly. More neurons = more complex boundaries. More layers = more levels of abstraction.

Output Layer

Produces the final prediction. For binary classification: 1 neuron with sigmoid. For multi-class: N neurons with softmax.

A network with one hidden layer of N neurons can approximate any continuous function (Universal Approximation Theorem). In practice, deeper networks with fewer neurons per layer often work better than shallow-wide networks — depth enables hierarchical feature learning.

MLP Computation

h(1) = σ(W(1)x + b(1))
h(2) = σ(W(2)h(1) + b(2))
…
y = σout(W(L)h(L-1) + b(L))

10.2 The Forward Pass

The forward pass is the computation that flows from input to output. Data enters at the input layer, gets transformed by each hidden layer’s weights and activation functions, and produces a prediction at the output.

For each layer l, the computation is:

Layer l Computation

z(l) = W(l)a(l-1) + b(l)

a(l) = σ(z(l))

Where z is the pre-activation (weighted sum) and a is the activation (after nonlinearity). We save both values at each layer because backpropagation needs them.

10.3 Loss Functions

The loss function measures how wrong the network’s prediction is. Training aims to minimize this number. The choice of loss function shapes what the network learns:

Mean Squared Error

L = ½ ∑(yi − ŷi)²

The default for regression. Penalizes large errors quadratically. Gradient is proportional to the error: simple and well-behaved.

Binary Cross-Entropy

L = −[y log ŷ + (1−y) log(1−ŷ)]

The default for binary classification. With sigmoid output, the gradient simplifies beautifully to (ŷ − y).

Categorical Cross-Entropy

L = −∑ yk log ŷk

For multi-class classification with softmax output. Used in virtually every language model — predicting the next token is a classification problem over the vocabulary.

10.4 The Chain Rule

Backpropagation is just the chain rule of calculus applied systematically. If we have a composite function f(g(h(x))), the derivative with respect to x is:

Chain Rule

∂f/∂x = (∂f/∂g) · (∂g/∂h) · (∂h/∂x)

A neural network is exactly such a composition: each layer is a function, and the full network is a chain of functions. The loss depends on the output, which depends on the last layer’s weights, which depend on the previous layer’s activations, and so on.

A computational graph represents the network as a directed acyclic graph (DAG) where each node is an operation and edges carry intermediate values.

For a simple two-layer network with loss L:

x → z(1) = Wx + b → a(1) = σ(z(1)) → z(2) = Va + c → ŷ = σ(z(2)) → L

The forward pass computes left-to-right, storing intermediate values. The backward pass computes right-to-left:

∂L/∂V = (∂L/∂ŷ) · (∂ŷ/∂z(2)) · (∂z(2)/∂V) = δ(2) · a(1)T

∂L/∂W = (∂L/∂ŷ) · (∂ŷ/∂z(2)) · (∂z(2)/∂a(1)) · (∂a(1)/∂z(1)) · (∂z(1)/∂W) = δ(1) · xT

Each δ is computed from the next layer’s δ — that’s why it’s called backpropagation. The error signal flows backward through the network.

10.5 Backpropagation

The backpropagation algorithm (Rumelhart, Hinton, Williams, 1986) trains a neural network in four steps, repeated for every training example:

1

Forward Pass

Compute activations layer by layer from input to output. Store all intermediate values (z and a at each layer).

2

Compute Loss

Compare prediction to target using the loss function. This single number encodes “how wrong” the network is.

3

Backward Pass

Compute gradients ∂L/∂w for every weight by propagating deltas backward through the chain rule.

4

Update Weights

w ← w − η · ∂L/∂w. Each weight is nudged in the direction that reduces the loss.

For a network with L layers, define the error signal (delta) at each layer:

Output Layer Delta

δ(L) = (∂L/∂a(L)) ⊙ σ'(z(L))

Hidden Layer Delta (Backprop Rule)

δ(l) = (W(l+1)T δ(l+1)) ⊙ σ'(z(l))

Weight Gradients

∂L/∂W(l) = δ(l) a(l-1)T

∂L/∂b(l) = δ(l)

The ⊙ is element-wise multiplication (Hadamard product). Notice that each layer’s delta depends on the next layer’s delta multiplied by the transposed weight matrix — this is the “backward” flow that gives the algorithm its name.

Complexity: One forward + backward pass takes O(W) time where W is the total number of weights. This is the same as one forward pass — backprop is essentially “free” relative to inference.

Key Insight

Backpropagation does NOT search for weights. It computes exact gradients — the precise direction in which each weight should change to reduce the loss. The only approximation is using a finite learning rate step instead of an infinitesimal one. This efficiency is why networks with billions of parameters can be trained at all.

10.6 Bridge: Cascade Systems in DSP

DSP → AI Connection

A multi-layer neural network is a cascade of nonlinear systems. In DSP, we analyze cascaded linear systems by multiplying their transfer functions. In AI, we analyze cascaded nonlinear layers by multiplying their Jacobian matrices.

DSP: Cascaded Filters AI: Multi-Layer Network
H(z) = H1(z) · H2(z) · … y = fL(fL-1(…f1(x)))
Transfer functions multiply Jacobians multiply (chain rule)
Poles accumulate → instability risk Gradient products → vanishing/exploding risk
Biquad sections for stability Residual connections for gradient flow
LMS adaptive filter: w += μ·e·x SGD update: w −= η·∇L

The analogy between cascaded filter stability and gradient flow through deep networks is deep: poles with |p| > 1 cause filter instability; Jacobian eigenvalues > 1 cause exploding gradients. Both are solved by keeping the system gains near unity — in DSP through careful pole placement, in AI through residual connections and normalization layers.

10.8 Tool: Backprop Step-Through

Walk through a single forward-backward pass on a tiny 2→2→1 network with exact arithmetic. Every multiplication, every gradient, every weight update is shown explicitly. This is the best way to truly understand what backprop does.

Tool 10.2 Backprop Step-Through
Try This
  • Click Next Step repeatedly to walk through all 10 steps
  • Steps 1–5 are the forward pass (cyan progress bar): data flows input → output
  • Steps 6–10 are the backward pass (orange): gradients flow output → input
  • Watch the delta (δ) values appear below each neuron — these are the local error signals
  • In steps 9–10, see the weight updates in green (Δ) along each connection
  • Notice how the output delta is large (close to target-prediction error) while hidden deltas are smaller — this is the vanishing gradient effect
