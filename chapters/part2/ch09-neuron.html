Chapter 09

The Neuron and the Perceptron

Every deep learning architecture — from a simple classifier to GPT — is built from one fundamental unit: the artificial neuron. It computes a weighted sum, applies a nonlinearity, and outputs a number. If that sounds like a matched filter with a threshold, you’re already seeing the connection.

9.1 From Biology to Math

A biological neuron receives electrical signals from thousands of other neurons via dendrites, sums them in the cell body, and if the total exceeds a threshold, fires an electrical pulse down its axon to other neurons.

In 1943, McCulloch and Pitts proposed a mathematical model that captures this behavior with just three operations:

1

Weigh

Each input xi is multiplied by a weight wi. Strong connections have large weights; inhibitory connections have negative weights.

2

Sum

Add all weighted inputs plus a bias: z = ∑ wixi + b. This produces the neuron’s “pre-activation” value.

3

Activate

Apply a nonlinear function: y = σ(z). Without this nonlinearity, stacking layers would be pointless — the whole network would collapse to a single linear transformation.

9.2 The Artificial Neuron

Neuron Computation

z = wTx + b = w1x1 + w2x2 + … + wnxn + b

y = σ(z)

The weighted sum z = wTx is a dot product — the same operation at the heart of convolution, correlation, and every linear system we studied in Part I.

In vector notation, the neuron computes:

y = σ(wTx + b)

Where:

  • x ∈ ℝn — input vector
  • w ∈ ℝn — weight vector (learned parameters)
  • b ∈ ℝ — bias (learned)
  • σ: ℝ → ℝ — nonlinear activation function

The dot product wTx = ‖w‖ ‖x‖ cosθ measures the alignment between the input and the weight vector. Maximum output occurs when x points in the same direction as w.

This is precisely the definition of a matched filter from DSP: the filter that maximizes the output SNR when the input matches the template.

What the Bias Does

The bias b shifts the decision boundary away from the origin. Without it, the neuron’s decision boundary must pass through the origin — a severe limitation. With bias, the neuron can separate any linearly separable dataset regardless of position.

9.3 Activation Functions

The activation function σ is what makes neural networks neural. Without it, every neuron is just a linear function, and a chain of linear functions is still linear. The activation introduces the nonlinearity that gives networks their power.

Step

The original: output 0 or 1. Binary decision. Problem: not differentiable, so we can’t compute gradients.

Sigmoid

Smooth S-curve mapping ℝ → (0,1). Differentiable everywhere. Problem: saturates for large |z|, causing vanishing gradients.

Tanh

Like sigmoid but zero-centered: ℝ → (−1,1). Better gradient properties but still saturates.

ReLU

max(0, z). The modern default. No saturation for z > 0, but “dead neurons” when z < 0 permanently.

GELU

Gaussian Error Linear Unit. Smooth approximation of ReLU used in Transformers (GPT, BERT). z · Φ(z).

Swish

z · σ(z). Self-gated, non-monotonic. Discovered by neural architecture search. Used in EfficientNet.

Without activation functions, a two-layer network computes:

y = W2(W1x + b1) + b2 = W2W1x + W2b1 + b2 = W’x + b’

This is just another linear transformation! No matter how many layers you stack, without nonlinearity the network can only learn linear decision boundaries. The Universal Approximation Theorem states that a single hidden layer with a nonlinear activation can approximate any continuous function to arbitrary accuracy (given enough neurons).

9.4 The Perceptron

The perceptron (Rosenblatt, 1958) is a single neuron with a step activation, trained to classify inputs. It learns by a simple rule: if the prediction is wrong, adjust the weights toward the correct answer.

Perceptron Learning Rule

w[t+1] = w[t] + η(y − ŷ)x

Where:

  • y is the true label (0 or 1)
  • ŷ is the predicted label
  • η is the learning rate

The Perceptron Convergence Theorem guarantees that if the data is linearly separable, the algorithm will find a solution in a finite number of steps. This was a landmark result — a machine that provably learns.

The perceptron defines a hyperplane in input space:

wTx + b = 0

In 2D, this is a line: w1x1 + w2x2 + b = 0. The weight vector w is the normal to this line. Points on one side (where wTx + b > 0) are classified as class 1; the other side as class 0.

The perceptron learning rule rotates and translates this hyperplane until all training points are correctly classified. Each misclassified point pulls the hyperplane toward the correct classification.

9.5 The XOR Problem

In 1969, Minsky and Papert published Perceptrons, proving that a single perceptron cannot learn XOR. The four XOR points require a nonlinear boundary — no single line can separate the classes:

x1 x2 XOR
000
011
101
110

This result triggered the first “AI winter” — funding dried up because people conflated the limitation of a single neuron with the limitation of neural networks as a whole. The fix is simple: add more layers. Two neurons can each draw a line, and a third can combine them to solve XOR. We’ll see this in Chapter 10.

9.6 Bridge: The Neuron IS a Matched Filter

DSP → AI Connection

The artificial neuron performs the same computation as a matched filter followed by a threshold detector — the fundamental building block of radar, sonar, and communication systems.

DSP: Matched Filter AI: Neuron
Template signal h[n] Weight vector w
Input signal x[n] Input features x
Cross-correlation ∑ h[k]x[k] Dot product wTx
Threshold detector Activation function σ(z)
Fixed template (designed) Learned weights (trained)
Max output when signal matches template Max output when input aligns with weights

The key difference: in DSP, the template h[n] is designed by an engineer who knows what signal to look for. In AI, the weights w are learned from data — the network discovers what patterns are important. This shift from designed to learned templates is one of the fundamental transitions from classical DSP to modern AI.

Looking Ahead

In Chapter 14, we’ll see that attention’s Q·KT operation is just a bank of learned matched filters running in parallel. The neuron you’re exploring here is the atomic unit of that entire mechanism.
